## 项目介绍

华溪互联网医院系统，包括与HIS对接，智慧医院、远程诊疗、网上药房、电子处方流转等子系统，满足各级别医院开展互联网诊疗；

提供全套互联网医院系统源码：总管理WEB端、医生WEB端、医生APP端、用户APP端、药房WEB端、药师APP端；

可对互联网医院系统进行功能定制开发或二次开发；
